# Python Task Agent Demo

Projeto simples de agente autonomo em Python para portfolio.

## Ideia

O agente recebe uma tarefa, cria uma lista de passos, executa funcoes locais e salva um log em JSON.

## Como executar

```bash
python main.py
```

## Estrutura

```text
main.py
memory/
```

## Saida

```text
memory/log.json
```
