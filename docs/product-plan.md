# AutoFin-AI Product Plan

## Goal

Turn AutoFin-AI into a clear portfolio project for financial review, structured calculations and JSON reports.

## Recommended first MVP

Build the core workflow before building a full app.

1. Input financial fields.
2. Validate required data.
3. Run deterministic calculations.
4. Generate a review log.
5. Export JSON and CSV reports.
6. Add a small FastAPI endpoint.

## Product ideas

### AutoFin Core

Python workflow for financial review and report generation.

### Invoice Copilot

Assistant that organizes invoice fields and prepares a review checklist before final user confirmation.

### Turnover Inspection App

Mobile checklist for property inspections with photos, timestamp, location metadata and exportable reports.

## Priority

1. AutoFin Core.
2. FastAPI demo.
3. Flutter or web prototype.
4. Inspection app prototype.
5. Advanced integrations after market validation.

## Portfolio positioning

This is a technical assistant for structured review and documentation, not a replacement for professional judgment.
